# 第2部 型に関する解析

「`"hello" + 1` はエラー」「`配列[文字列]` はおかしい」── こうした判断を、プログラムを**実行する前に**下せるとうれしい。それを可能にするのが **型（type）** にまつわる解析です。

第2部では、まず型の正しさを検査する **型検査（type checking）** を、型システムと型規則という枠組みを通して学びます。次に、プログラマが型を書かなくても処理系が型を**推測してくれる** **型推論（type inference）** を扱い、関数型言語の心臓部であるヒンドリー・ミルナー型推論まで踏み込みます。型は、意味解析のなかでもとりわけ理論が美しく整備された分野です。
